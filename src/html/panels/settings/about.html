About Wordmark

Version

Wordmark is an open source AI assistant that uses the Responses API with OpenAI, xAI or LM Studio. It offers advanced features like personality customization, tool calling (weather, OpenAI web search, and custom MCP servers), text-to-speech, theme customization, and secure local storage of conversations and API keys.

All of your conversations and generated images are stored locally in your browser. You can easily manage your chat history and view your images in the image gallery. Make sure to download the images you want to keep, as they are not stored on any server and will be lost if you clear your browser data.

For more information, visit the GitHub Repository.
